Internship Project

EPSCO Arc Flash Study Tool

A mobile application built for electrical engineers to conduct Arc Flash studies, input system data, and quickly assess factory safety compliance — developed at Icebreaker Games for EPSCO Inc.

Unity C#

My Role

Contributed to the development of the application as a Software Engineering Intern at Icebreaker Games. Worked on UI components and core application logic within a Unity and C# codebase.

Project Type

Client-contracted mobile application developed at Icebreaker Games for EPSCO Inc., a provider of electrical engineering services.

Screenshot

Overview

Arc Flash studies are a required part of electrical safety compliance in industrial environments. Engineers must analyze electrical system parameters to determine incident energy levels, establish arc flash protection boundaries, and ensure workers are equipped with appropriate PPE. This tool was built to bring that analysis workflow to a mobile device, allowing engineers to input system data on-site and receive results quickly rather than relying on desktop software back at the office.

Challenges & Problems Solved

  • Built a data-driven input system in Unity and C# capable of handling the structured numerical inputs required for Arc Flash calculations, with validation to prevent incorrect entries.
  • Designed a UI in Unity suited for professional use in the field, balancing information density with readability on mobile screens.
  • Worked within a client-driven development environment at Icebreaker Games, adapting to requirements and feedback from EPSCO's engineering team throughout the project.
