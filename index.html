About

I'm a software engineering graduate from St. Cloud State University with hands-on experience building full-stack web applications, machine learning pipelines, desktop tools, and mobile apps used in real-world professional contexts. My projects have ranged from automating asset detection in construction models to developing safety analysis software for electrical engineers — work that pushed me to pick up new technologies quickly and deliver across the whole stack.

I'm drawn to problems where software can create real, tangible impact, whether that's improving an engineer's daily workflow, bringing a machine learning model into a usable product, or crafting a mobile experience people actually enjoy. I'm comfortable moving between frontend and backend, and I approach each project as an opportunity to learn something new.

Outside of professional work, I've spent years building games in Unity and modeling in Blender — a hobby that has sharpened my instincts for user experience and interactive design. I've also been playing guitar since I was 10, which has given me a deep appreciation for the kind of patience and long-term commitment that serious craft requires.
